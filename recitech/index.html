Como ganhar pontos?

Você ganha pontos realizando ações sustentáveis como:

  • Reciclar materiais (+20 pontos)
  • Fazer denúncias ambientais (+30 pontos)
  • Participar de eventos e mutirões (+80 pontos)
  • Completar desafios semanais (+50 pontos)
Como fazer uma denúncia?

Para fazer uma denúncia ambiental:

  1. Toque no botão "Denúncia Verde" no menu
  2. Selecione o tipo de problema ambiental
  3. Tire fotos do local
  4. Descreva o problema em detalhes
  5. Envie a denúncia para análise
Como usar o calendário?

O calendário mostra os dias de coleta seletiva na sua região:

  • Cores diferentes representam tipos de coleta
  • Toque em um dia para ver detalhes da coleta
  • Use os filtros para ver tipos específicos de material
  • Ative lembretes para não perder a coleta
